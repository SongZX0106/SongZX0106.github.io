# HexoBlog

## 介绍

使用 hexo + butterfly 搭建的个人博客

## 预览地址

https://szxio.gitee.io/hexoblog/

## 安装教程

```shell
npm install
```

## 使用说明

1. 全局安装 hexo

```shell
npm install hexo-cli -g
```

2. 安装项目依赖

```shell
npm install
```

3. 启动

```shell
hexo server
```

4. 清除缓存

```shell
hexo clean
```
